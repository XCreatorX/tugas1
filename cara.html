Cara Root Android Download aplikasi Unlock Root v3.41 --> download Ket: Sampai artikel ini di tulis, Software ini support Android 2.2-2.3, 4.0.0-4.0.3 (detailnya lihat keterangan di websitenya), dan hanya dapat di install pada komputer dengan OS Windows 7/Vista/XP.
1.Aktifkan mode USB debugging di Ponsel anda melalui menu “Setting > Applications > Development > centang USB debugging”.

2.Hubungkan perangkat Android anda dengan PC menggunakan kabel USB

3.Jalankan aplikasi “UnlockRoot.exe” yang telah di download tadi, dan tunggu sampai software tersebut selesai mengenali jenis perangkat Android anda (dalah hal ini akan terdeteksi jika jenis perangkat Android sudah di support).

4.Pilih perangkat dari daftar perangkat yang di deteksi.

5.Klik tombol Root untuk memulai proses Rooting.

6.Tunggu hingga proses selesai, dan jika berhasil maka akan muncul “your device has been successfully rooted”.

7.Setelah selesai, Reboot perangkat android anda agar perubahan diterapkan.
